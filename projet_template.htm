Dashboard Power BI - Analyse des Ventes

Contexte

Développement d'un dashboard interactif pour une entreprise de retail, permettant de suivre les performances de vente en temps réel.

Technologies Utilisées

  • Power BI
  • DAX
  • SQL

Fonctionnalités Clés

  • Analyse des ventes par région
  • Prévisions de revenus
  • Identification des produits les plus performants
