Dashboard Power BI - Analyse des Ventes

Power BI SQL Data Visualization

Contexte

Projet d'analyse des ventes pour une entreprise de retail, visant à améliorer la compréhension des performances commerciales.

Objectifs

  • Créer un dashboard interactif
  • Analyser les tendances de vente
  • Fournir des insights exploitables

Technologies Utilisées

Power BI DAX SQL

Résultats

Le dashboard a permis de :

  • Réduire les coûts de 15%
  • Améliorer la prise de décision
  • Visualiser clairement les performances
